# Kolektt SNS Templates

이 폴더에는 SNS 템플릿 PSD 파일들이 저장됩니다.

## 파일 구조

```
templates/
├── feed-template-1-text.psd (1080x1350px)
├── feed-template-2-product.psd (1080x1350px)
├── feed-template-3-announcement.psd (1080x1350px)
├── feed-template-4-cardnews.psd (1080x1350px)
├── story-template-1-bts.psd (1080x1920px)
└── story-template-2-feature.psd (1080x1920px)
```

## 템플릿 제작 가이드

### Feed Posts (4:5 - 1080x1350px)
- 좌측 정렬
- Ivory 배경 (#f6f6ed)
- Plus Jakarta Sans (제목), Pretendard (본문)
- 블루 액센트는 포인트로만 사용

### Story Posts (9:16 - 1080x1920px)
- 조용하고 디테일한 톤
- 미니멀 레이아웃
- 여백 충분히 확보

## 사용 방법

1. 포토샵에서 위의 사이즈로 템플릿 제작
2. 파일명 규칙에 맞춰 이 폴더에 저장
3. 웹사이트에서 자동으로 다운로드 링크 표시됨
